--- title: "TextMate & Komodo" date: Mon Jun 25 2007 00:00:00 GMT+1200 (New Zealand Standard Time) ---

Wow. TextMate still blows me away with its features and bundles. It's the best value software I ever bought. I just discovered some more great features with SVN and CSS bundles on this pretty site:

  • http://blog.circlesixdesign.com/2006/10/20/textmate-css-editing/
  • http://blog.circlesixdesign.com/2007/04/18/textmate-svn-bundle/

I've also been using Komodo IDE for debugging for the last week. It's great. A BIG time-saver when you're sure your code is right, but it's still not behaving as you expected. It's pricey, but worth it if you spend more than a few hours a week going through your code line by line to find what went wrong. I'm eligible for education pricing, so I got it for $50 USD.
